# Install the extension

## Download

Go to the [latest release](https://github.com/yourusername/binge-watch-me-extension/releases/latest)
and download `BingeWatchMe-extension.zip`.

## Load in browser

1. Unzip `BingeWatchMe-extension.zip`
2. Open `brave://extensions` or `chrome://extensions`
3. Enable **Developer mode** (top right toggle)
4. Click **Load unpacked**
5. Select the unzipped extension folder

## Connect to the daemon

1. Make sure `BingeWatchMe.app` is running (`BWM` visible in menu bar)
2. Click the `BWM` extension icon in your browser toolbar
3. Click **Connect to daemon**
4. You should see **Connected!**

The extension reconnects automatically every time the daemon starts.
