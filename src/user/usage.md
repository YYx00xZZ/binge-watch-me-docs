# Using the remote

Open Netflix in your browser and start watching. Your phone remote will show
the current title and playback controls.

## Controls

| Button | Action |
|---|---|
| Play / Pause | Toggle playback |
| Next episode | Skip to next episode |
| Vol + | System volume up |
| Vol - | System volume down |

## Volume

Volume buttons control your Mac's system volume, not Netflix's internal volume.

## Progress bar

The progress bar and timestamps update every second from the actual video position.
